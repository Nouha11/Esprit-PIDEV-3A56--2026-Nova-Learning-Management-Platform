## XP Progression Formula

The XP requirements follow a progressive formula where each level requires approximately 100-350 more XP than the previous level. This creates a smooth difficulty curve that:

- Allows new players to level up quickly (Levels 1-10)
- Provides steady progression for intermediate players (Levels 11-30)
- Creates meaningful milestones for advanced players (Levels 31-50)
- Offers prestige levels for dedicated players (Levels 51-60)

## Rank Name Changes

Rank names change every 5 levels to provide clear progression milestones:

1. **Novice** (Levels 1-4): Beginners just starting their journey
2. **Apprentice** (Levels 5-9): Learning the basics
3. **Skilled** (Levels 10-14): Developing competence
4. **Adept** (Levels 15-19): Showing proficiency
5. **Professional** (Levels 20-24): Consistent performance
6. **Expert** (Levels 25-29): High-level mastery
7. **Veteran** (Levels 30-34): Experienced players
8. **Elite** (Levels 35-39): Top-tier performers
9. **Master** (Levels 40-44): Near-perfect execution
10. **Grandmaster** (Levels 45-49): Exceptional skill
11. **Champion** (Levels 50-54): Tournament-level play
12. **Legend** (Levels 55-59): Legendary status
13. **Mythic** (Level 60+): Ultimate achievement

## Display Locations

The rank name is now displayed in:

1. **Student Dashboard** (`/student/dashboard`)
   - Under the profile header: "Level X - Rank Name"
   - In the stats card: Shows rank name instead of "Current Level"

2. **My Rewards Page** (`/rewards/my-rewards`)
   - In the Total XP card: "Level X - Rank Name"

3. **Leaderboard** (if implemented)
   - Next to each player's level

## Level Milestones

Admins can create Level Milestone rewards that unlock automatically when students reach specific levels. These rewards:

- Are managed in the admin panel under Rewards → Level Milestones
- Award bonus tokens automatically when the level is reached
- Display in the Rewards Gallery under "Level Milestones" filter
- Show a celebration modal when unlocked

## Implementation Details

**Service**: `src/Service/game/LevelCalculatorService.php`
- Contains all level thresholds and calculations
- Provides methods for level names, colors, icons, and progress

**Methods**:
- `calculateLevel(int $xp)`: Returns level info including name, progress, etc.
- `getLevelName(int $level)`: Get rank name for a specific level
- `getLevelBadgeColor(int $level)`: Get Bootstrap color class
- `getLevelIcon(int $level)`: Get Bootstrap icon class

**Usage Example**:
```php
$levelInfo = $levelCalculator->calculateLevel($student->getTotalXP());
// Returns: ['level' => 15, 'name' => 'Adept', 'progress' => 45.2, ...]
```
