# Game Categories Implementation - Full Game vs Mini Game

## Summary
Implemented a game category system that distinguishes between "Full Games" (with rewards) and "Mini Games" (energy regeneration only). This prepares the system for future energy and course integration.

## Game Categories

### FULL_GAME
- **Purpose**: Traditional games with complete reward system
- **Features**:
  - Token cost to play
  - Reward tokens upon completion
  - Reward XP upon completion
  - Can have special rewards (badges, achievements, bonus XP/tokens)
  - Tracked in leaderboards
  - Full gamification experience

### MINI_GAME
- **Purpose**: Energy regeneration games for future course/energy system
- **Features**:
  - Token cost to play (optional)
  - Energy points regeneration (configurable by admin)
  - NO traditional rewards (tokens, XP, badges)
  - Designed to be linked with courses later
  - Quick, simple gameplay focused on energy recovery

## Database Changes

### New Fields in `game` Table

```sql
ALTER TABLE game 
ADD category VARCHAR(50) NOT NULL DEFAULT 'FULL_GAME',
ADD energy_points INT DEFAULT NULL;
```

#### `category` Field
- Type: VARCHAR(50)
- Values: 'FULL_GAME' or 'MINI_GAME'
- Default: 'FULL_GAME'
- Not nullable
- Determines game behavior and available features

#### `energy_points` Field
- Type: INT
- Nullable: YES
- Purpose: Amount of energy points regenerated by mini games
- Only applicable for MINI_GAME category
- Can be modified by admin

## Admin Interface Changes

### Create/Edit Game Forms

#### Dynamic Field Visibility
JavaScript automatically shows/hides fields based on selected category:

**When FULL_GAME is selected:**
- ✅ Show: Reward Tokens field
- ✅ Show: Reward XP field
- ✅ Show: Special Rewards section
- ❌ Hide: Energy Points field

**When MINI_GAME is selected:**
- ✅ Show: Energy Points field
- ❌ Hide: Reward Tokens field
- ❌ Hide: Reward XP field
- ❌ Hide: Special Rewards section

#### JavaScript Implementation
```javascript
const categorySelect = document.querySelector('[data-game-category="true"]');
const energyField = document.getElementById('energy-points-field');
const fullGameRewards = document.getElementById('full-game-rewards');

function toggleFields() {
    const category = categorySelect.value;
    
    if (category === 'MINI_GAME') {
        energyField.style.display = 'block';
        fullGameRewards.style.display = 'none';
    } else {
        energyField.style.display = 'none';
        fullGameRewards.style.display = 'block';
    }
}

categorySelect.addEventListener('change', toggleFields);
```
